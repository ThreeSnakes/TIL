### 정규 표현식이란

- 정규 표현식은 특정 문자 조합과 대응하기 위해 사용되는 패턴을 말한다
- 문자열의 검색과 치환을 위해 주로 사용된다

### 사용법

#### 정규식 생성방법

```js
// 리터럴을 이용하는 방법
const regex = /ab+c/;

// 생성자를 이용하는 방법
const regex2 = new RegExp("ab+c");
```
